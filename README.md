# 使用LLM、langchain、fastapi、agent等技术实现ai chat agent

## 入口详见：@app.post("/chat/sse")

## 相关功能：
* llm和用户聊天
* sse流式输出
- 支持fastapi接口
- 支持本地向量库
- 使用langchain 生成ai agent
- 支持conversation agent流式返回中文回答
